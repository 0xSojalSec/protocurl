  protocurl [flags] -f proto-file -i request-type -o response-type -u url -d request-text

If no curl executable was found in the path, it will fall back to an internal non-configurable http request.

Examples:
  protocurl -I my-protos -f messages.proto -i package.path.Req -o package.path.Resp -u http://foo.com/api -d "myField: true, otherField: 1337"
